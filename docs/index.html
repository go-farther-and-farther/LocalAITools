功能一览
一套工具解决图片整理、文本处理、知识管理的常见需求
🤖

AI 智能分类

勾选分类 → AI 自动归类图片到对应文件夹。智能分组加速，1000 张相似截图只需几十次 API 调用。

🖼️

图片重命名

6 种命名模式（通用/人像/风景/截图/美食/动漫），AI 生成中文短句替代杂乱文件名。试运行结果一键应用。

🔍

质量评分

7 种检测模式覆盖 AI 图筛选、漫展选片、人像评估、文档扫描等场景。评分画廊 + CSV 导出。

💬

聊天

多对话管理、流式输出、思考过程可见、联网搜索、RAG 知识库检索。Enter 发送，自动保存。

📝

聊天压缩 & 翻译

OCR 聊天记录精简压缩。长篇文本按章节翻译，术语表保证一致性，断点续传。

📚

知识库问答

上传文档 → 向量索引 → 对话式检索。支持 PDF/DOCX/HTML/URL，混合检索（向量 + BM25）。
