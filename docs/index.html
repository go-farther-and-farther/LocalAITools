功能一览
一套工具解决图片整理、文本处理、知识管理的常见需求
🤖

AI 智能分类

勾选分类（照片/动漫/游戏/绘画/截图/风景/美食/文档），AI 自动归类。智能分组加速，1000 张相似截图只需几十次 API 调用。

⚡ 智能分组 · 试运行保存
🖼️

图片重命名

6 种命名模式（通用/人像/风景/截图/美食/动漫），AI 生成中文短句替代杂乱文件名。支持自定义提示词。

试运行结果一键应用
🔍

质量评分

7 种检测模式：AI 图错误检测、漫展摄影筛选、人像评估、风景评估、文档扫描、绘画插图。评分画廊 + CSV 导出。

自动分拣高/低质量
💬

AI 聊天

多对话管理、流式输出、思考过程可见。AI 自主联网搜索（最多 3 轮），RAG 知识库检索可调。

联网搜索 · 知识库
📝

聊天压缩 & 翻译

OCR 聊天记录精简压缩。长篇文本按章节翻译，术语表保证一致性，断点续传，多线程并行。

术语表 · 断点续传
📚

知识库问答

上传文档 → 向量索引 → 对话式检索。支持 PDF/DOCX/HTML/URL，混合检索（向量 + BM25）。

FAISS · BM25 混合检索
⚡

LLM 压测

测试 API 推理性能：TTFT、ITTL、吞吐量。自动生成专业图表报告，历史记录对比。

自动图表 · CSV 导出
