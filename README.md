# 📸 SnapshotZen - 图片智能中文命名工具

让本地大模型帮你“看懂”截图、照片，一键生成 **5~25 字优美中文短句**，并按时间顺序自动重命名。

> 🎯 告别 `屏幕截图2026-04-11_005848.png`，迎来 `《原神》璃月夜景_2026-04-11 005848.png`

---

## ✨ 功能亮点

- **🧠 多模态理解**：调用本地视觉模型（LM Studio），准确识别截图内容
- **📚 上下文记忆**：记住最近处理的 5 张图片描述，保持命名风格一致（尤其适合连续截图、系列作品）
- **🕒 时间排序**：自动从文件名提取时间戳，未含时间的图片回退到文件修改时间，按时间先后处理
- **⚡ 并行加速**：多线程并发调用模型，批量处理更快
- **🧪 安全试运行**：`--dry_run` 模式只预览不会实际修改文件
- **🎮 作品名识别**：能识别游戏、动漫、电影等知名 IP，自动加上 `《》` 书名号

## 🧰 环境准备

### 1️⃣ 安装 LM Studio 并加载视觉模型
- 下载 [LM Studio](https://lmstudio.ai/)（支持 Windows / macOS / Linux）
- 在 "Search" 中搜索支持视觉的模型，推荐：
  - `qwen/Qwen2.5-VL-7B-Instruct` 或 `qwen/Qwen3.6-27b`
- 下载后启动本地服务器（默认 `http://localhost:1234`）

### 2️⃣ 克隆仓库 & 安装依赖

git clone https://github.com/你的用户名/SnapshotZen.git
